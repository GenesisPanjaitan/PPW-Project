# PPW-Project

CareerConnect adalah platform berbasis web yang dirancang untuk membantu mahasiswa dan alumni Institut Teknologi Del dalam mengakses serta berbagi informasi peluang karier berupa lowongan kerja dan magang.

Mahasiswa dapat menelusuri dan menyimpan lowongan yang diminati, sementara alumni dapat mempublikasikan informasi lowongan dari perusahaan tempat mereka bekerja. Sistem ini dilengkapi dengan pengelolaan profil pengguna, fitur favorit, serta struktur data terintegrasi untuk mendukung proses pencarian karier secara efektif.
